# BlogDemo.
### Веб-сайт блога с категориями и постами

## Инструкция к установке
1) Клонируем репозиторий в машину

``
git clone https://github.com/AutoPilot88/BlogDemo.git
``

2) Заходим в папку с проектом

``
cd BlogDemo
``

3) Подготавливаем файлы .env

``
cp .env.example .env
``

4) Запускаем docker контейнеры

``
docker-compose up -d
``
 
5) Заходим по установленному порту на браузере (дефолтный 80)


## Технологический стек:
- PHP 8.2 
- Шаблонизатор Smarty
- SCSS 
- MySQL
- Nginx
- Docker

## Структура данных:
- Категория:
  - Название
  - Описание
- Публикация:
  - Изображение
  - Название
  - Описание
  - Текст
  - Категория (одна или несколько)
  - Кол-во просмотров

## Страницы:
- Главная страница:
  - Отображается все категории и 3 последних поста (по дате публикации).
  - Отображается кнопка “Все статьи” для каждой категории.
- Страница категории:
  - Отображается название, описание, список статей
  - Сортировка статей (по кол-во просмотров, по дате публикации)
  - Пагинация (появляется, если статей больше 9и)
- Страница статьи:
  - Отображается вся информация о статье
  - Отображается блок из 3 похожих статей

#### * По пути /docker/mysql/ находятся миграции и сиды
